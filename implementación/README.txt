Instrucciones:

-Para compilar, ejecutar desde el directorio "implementación" el comando:
 "make compile"
-Para ejecutar, ejecutar desde el directorio "implementación" el comando:
 "./main semilla archivo P(>0), E(entre 0 y 1), numeroGeneraciones, Pcross(entre 0 y 1), Pmut(entre 0 y 1), alpha(>1), W (mayor que 0. Opcional, si no se ingresa es igual a 9000)"

Indicaciones:

1. Los nombres de archivo que se ingresan en la ejecución no incluyen el directorio instancias. Es decir, en lugar de ingresar "instancias/NL10.txt" se ingresa simplemente como "NL10.txt". 
2. Los otros parámetros que se ingresan en la ejecución (excepto la semilla) se explican en el informe
